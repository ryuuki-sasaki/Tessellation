★説明
	距離適応型テッセレーション(Distance Adaptive Tessellation)に
	面の向き適応型テッセレーション(Orientaion Adaptive Tessellation)を
	組み合わせました。これによって、背景の山などの背景モデルと視点の距離、向きによって
	輪郭の詳細度を損なわず、内側の面の詳細度を下げることができます。
		
★マウス操作
	左クリックしながら操作
		カメラ回転
	右クリックしながら上下
		カメラのｚ方向の移動

★プログラムを組む上で工夫したこと
	視点に対する法線の角度に応じて面の詳細度を変化させる
	面の向き適応型テッセレーション(Orientaion Adaptive Tessellation)を実装しました。
